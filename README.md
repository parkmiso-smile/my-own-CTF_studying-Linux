# Vim Whitespace Steganography Challenge

이 저장소는 Vim 에디터의 숨겨진 문자 표시 기능을 활용해 제작한
화이트스페이스 기반 스테가노그래피(정보 은닉) CTF 문제를 담고 있습니다.

문제 파일은 평범한 텍스트 문서처럼 보이지만,  
마지막 줄에 특정 방식으로 인코딩된 비밀 메시지가 포함되어 있습니다.  
이는 일반적인 출력 방식으로는 보이지 않으며,  
문제 해결자는 적절한 도구를 사용해 숨겨진 정보를 식별해야 합니다.

문제의 배경은 토끼 왕국이 감청을 피하기 위해  
공백(Space)과 탭(Tab)을 조합한 은밀한 방식으로  
강아지 왕국에게 경고 메시지를 전달한다는 설정으로 이루어져 있습니다.

제공된 파일들에는 문제 생성에 사용된 스크립트와  
은닉 텍스트가 포함된 실제 문제 파일이 함께 포함되어 있습니다.
참가자는 파일 내용을 분석해 토끼 왕국이 남긴 메시지를 찾아내면 됩니다.
